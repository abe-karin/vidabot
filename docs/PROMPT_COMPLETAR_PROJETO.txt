CONTEXTO DO PROJETO:
Estou construindo um chatbot de seguros de vida chamado VidaBot para a empresa fictícia InsurMinds.
O projeto é uma entrega acadêmica de pós-graduação/MBA com tema: IA aplicada ao setor de seguros.

O que JÁ TENHO:
- Interface HTML completa (frontend/index.html): chat moderno navy/gold, RAG simulado com keywords,
  integração com Claude API (claude-sonnet-4-20250514), animação do fluxo RAG, painel de chunks,
  sidebar com atalhos e estatísticas em tempo real.
- Backend básico em Flask (app.py) com endpoint /api/chat e busca RAG por keywords no faq.txt
- Base de conhecimento (faq.txt) com 15 chunks sobre coberturas, sinistros, carências, VGBL/PGBL
- requirements.txt e README.md

STACK:
- Python 3.11, Flask, Anthropic SDK
- Frontend: HTML/CSS/JS puro (sem framework)
- RAG: busca por keywords (sem embeddings por ora)
- IA: Claude claude-sonnet-4-20250514
